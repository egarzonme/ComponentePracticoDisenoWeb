DISEÑO SITIOS WEB

BOX MODEL

es un concepto fundamental en CSS (Cascading Style Sheets) que describe cómo se calculan y aplican las dimensiones y las propiedades de un elemento HTML en una página web.

COMPONENTES

  • Contenido (Content): Es el área donde se muestra el contenido real del elemento, como texto, imágenes o videos. El tamaño del contenido se especifica mediante las propiedades width (ancho) y height (alto).
  • Relleno (Padding): Es el espacio transparente que se encuentra entre el contenido y los bordes de la caja. El relleno se puede especificar mediante las propiedades padding-top, padding-right, padding-bottom y padding-left. Estas propiedades controlan el espacio adicional dentro de la caja alrededor del contenido.
  • Borde (Border): Es una línea que rodea el contenido y el relleno. El borde se puede especificar mediante la propiedad border, que incluye la anchura (border-width), el estilo (border-style) y el color (border-color). Los valores comunes para el estilo del borde incluyen solid (sólido), dotted (punteado) y dashed (discontinuo).
  • Margen (Margin): Es el espacio transparente que se encuentra fuera del borde y separa un elemento de otros elementos adyacentes. Los márgenes se pueden especificar mediante las propiedades margin-top, margin-right, margin-bottom y margin-left. Estas propiedades controlan el espacio entre las cajas de los elementos.
